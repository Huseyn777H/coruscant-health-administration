Main features

  • Patients can register and upload daily health readings.
  • Doctors can create reports and request services like CT scan or PET scan.
  • Departments can see assigned orders and upload results.
  • Emergency users can quickly add new patient intake data.

Security

  • Users have different roles in the system.
  • Patient and doctor accounts need admin approval.
  • Uploaded files are encrypted before being stored.
  • The project can be deployed with cloud settings.
